# Mockup: Inicio + Login Seguro (Chips)

Versión alternativa del mockup con selector de perfil mediante "galletas" (chips). Usa la misma paleta de colores y comportamientos simulados.

## Perfiles
- Usuario (comprar y vender) — por defecto, sin aprobación.
- Compraventa — requiere aprobación tras autenticación.
- Concesionario — requiere aprobación.
- Taller — requiere aprobación.

## Cómo probar
1. Abre `index.html`.
2. Marca el consentimiento para habilitar los botones.
3. Cambia el chip de perfil y prueba Login/Google/WhatsApp.
4. Observa el aviso de "pendiente de aprobación" en perfiles especiales.
